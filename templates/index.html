1. OpenWeatherMap Key

You need a free API Key for weather data.
Register at: home.openweathermap.org
Create an "API Key" there.

2. Coordinates (Lat / Lon)

Required for accurate local weather.
Open Google Maps, right-click your location.
The first two numbers are Lat (Latitude) and Lon (Longitude).

3. Sensor ID (Optional)

For local air quality data (particulate matter).
Visit maps.sensor.community.
Click a sensor near you. The ID is at the top (e.g., 12345).

URL Structure

Copy this to your browser and replace the values:

?owm=YOUR_KEY&lat=51.5&lon=-0.1&sensor=ID
